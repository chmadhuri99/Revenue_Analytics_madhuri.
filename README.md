# 📊 Edviron Revenue Analysis

## 🔍 Project Overview
This project analyzes transactional data to calculate commission, settlement, and net revenue using Excel and Power BI.

## 🛠 Tools Used
- Excel
- Power BI
- DAX
- Power Query

## 📈 Key Insights
- 85% transactions via UPI
- Flat 2% merchant commission
- 12% missing capture status
- Revenue highest in April

## 🧮 Key Calculations
- Merchant Commission = Transaction Amount × 2%
- Partner Commission = Transaction Amount × 2%
- Net Revenue = Transaction - Commission - Vendor Amount
